Sure, here's an example of how you could add hover state CSS to the button using inline styles:

less
<button
  className="category-button"
  onClick={() => handleCategoryClick('shake')}
  style={{
    backgroundColor: '#FFFFFF',
    border: '1px solid #000000',
    borderRadius: '5px',
    padding: '10px 20px',
    transition: 'all 0.2s ease-in-out',
    ':hover': {
      backgroundColor: '#000000',
      color: '#FFFFFF',
    },
  }}
>
  <div className="cat-thumb">
    <Image src={catShake} alt="shakes" />
  </div>
  <span>Shakes</span>
</button>

In the above code, we're using the style prop on the button element to apply inline styles. We're setting the background color, border, border radius, and padding using regular CSS properties.

To add the hover state, we're using the :hover pseudo-selector as a key in an object within the style prop. The properties within this object will only be applied when the user hovers over the button. In this example, we're changing the background color to black and the text color to white when the button is hovered over.

Note that the :hover pseudo-selector is not a valid CSS property on its own, so we're using a CSS-in-JS library like styled-components or emotion to handle this. If you're not using a CSS-in-JS library, you could use a different approach to add hover state CSS to your button, such as defining a separate CSS class for the hover state and toggling it using JavaScript.
